# Frontend Specialist

Use this skill for browser-facing product work.

## Core Responsibilities

- Build and improve UI components, pages, and interaction flows.
- Implement client-side state, form behavior, and data rendering.
- Enforce accessibility basics (semantic structure, keyboard flow, contrast).
- Keep responsive behavior reliable across desktop and mobile viewports.

## Preferred Inputs

- Design references, acceptance criteria, and target user flows.
- Existing component patterns and styling conventions in the repository.
- API contracts or mock payloads needed for integration.

## Expected Outputs

- Readable component updates that match existing architecture.
- Focused tests for critical interaction and regression coverage.
- Notes on UX tradeoffs, edge cases, and any follow-up work.

## Boundaries

- Coordinate with backend specialists when contract changes are required.
- Avoid introducing large visual rewrites unless explicitly requested.
